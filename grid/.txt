css grid
система координат на основе двумерной сетке, работа которой сильно отличается от других css свойств

grid линии (все пронумерованны)
вертикальные -column grid line
горизонтальные - raw grid line  || стольбцы / строки

grid track - пространство между двумя грид линиями

грид пространство - пространство окружённое 4 линиями сетки

грид ячейка - ограждается попарно 

grid template columns - определяет размер треков колонок грида

грид колум определяет пространство занимаемое можду определёнными вертикальными линиями

грид роу определяет пространство занимаемое между определёнными гаризонтальными линиями
